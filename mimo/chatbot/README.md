# Chatbot Project (Mimo Practice)
## Project Overview
This project is a simple Python chatbot created as part of Mimo practice. The chatbot asks the user for their name, age, and favorite color, and then responds based on their answers. It is designed to help practice user input, variables, and basic arithmetic in Python.
## Features
- Asks for the user's name and gives a friendly response
- Calculates the difference between the user's age and the bot's age (3 years old)
- Responds to the user's favorite color
- Uses f-strings for cleaner and more readable output
